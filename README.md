## Project Purpose
Mobile app to help residents discover and register for local events. Staff can manage events efficiently. MVP uses local JSON data for testing and supports offline mode with cached data.

## Core Features
- View today’s and upcoming events
- Event search by keyword and category filter
- Event registration with success/error messages
- Navigation between Home, Events, Register, and Settings
- Home screen shortcut to “Today’s Events”
- Settings screen with theme toggle, font size adjustment, and sound toggle
- Logo click plays a sound (assets/sounds/sound.wav)
- Offline mode banner when server is unreachable
- Error message banners for failed API calls
- Event CRUD functions (add, update, delete) via API utilities

## Tech Stack
- React Native with Expo
- React Navigation (stack + tab navigators)
- React Native Paper UI components
- Local JSON / mock API data
- AsyncStorage for offline caching
- Git & GitHub for version control

# Elevate Horizon Connect

**Author:** Amir Sharabiani

**Student No:** 881472750


## Project Purpose
Mobile app to help residents discover and register for local events. Staff can manage events efficiently. MVP uses local JSON data for testing.

## Core Features
- View today’s and upcoming events
- Event search by keyword or date
- Event registration
- Navigation between Home, Events, Register, Settings

## Tech Stack
- React Native with Expo
- React Navigation
- Local JSON data
- Styled Components / StyleSheet
- Git & GitHub for version control

## Project Status
Planning & Setup – Local Expo project created, initial commit done, project plan drafted.
